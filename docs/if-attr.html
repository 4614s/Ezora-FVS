Dokümantasyon 2

_if Özniteliği: Kullanımı ve Sözdizimi

EzoraFVS'te _if özniteliğini bilmek çok önemlidir. Bu sayfada size _if özniteliğini anlatacağız. Bu sayfayı okumadan önce javascript operatörlerini ve string metotlarını bilmeniz gerekmektedir. Ayrıca dokümantasyonun giriş bölümünüde okumuş olmalısınız.

_if Sözdizimi:

_if özniteliğinde; $val, $len ve $checked olmak üzere üç özel değişken vardır

  • $val alanın yazı değeridir. (Tipi: String)
  • $len alanın yazı değerinin uzunluğudur. (Tipi: Number)
  • $checked kutucuğun işaretlenip işaretlenmediğidir. (Tipi: Boolean)

Operatörler

1. == İle != Operatörü

== operatörü değerin eşit olmasını kontrol eder.
!= operatörü değerin eşit olmamasını kontrol eder.
Bu iki operatör javascript içindekilerden farklıdır. Tip dönüşümü yoktur. 5 == "5" yanlış çıkar mesela javascriptteki == operatörü doğru derdi.

Diğer operatörler

>, <, >=, <=, !, ||, && ... Bu operatörler javascript içindeki kullanımı ile aynıdır.

String metotları

.lower ile .upper

_if içinde kullanımı javascriptteki karşılığı
x.lower x.toLowerCase()
x.upper x.toUpperCase()
x.lower(y) x.toLocaleLowerCase(y)
x.upper(y) x.toLocaleUpperCase(y)

Diğer metotlar

_if içinde kullanımı javascriptteki karşılığı
x.end(y) x.endsWith(y)
x.start(y) x.startsWith(y)

.split() ile .includes() kullanımı javascriptteki ile aynıdır.

Bu metotlar dışındaki hiçbir metot kullanımına izin verilmez.

Bilgi: _if özniteliği sözdizimi regex (Regular expressions) desteklemez.

$len >= 8 && $len <= 20 koşulunun uzunluk 8'e eşit veya 8'den büyük ve uzunluk 20'ye eşit veya 20'den küçük olmasıdır.
